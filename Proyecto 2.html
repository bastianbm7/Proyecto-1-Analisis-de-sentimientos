9.- Para el mismo libro, usando el léxico NRC, halle para cada emoción, los capítulos más representativos para cada una.¶

Para hacer esto, se generar una proporcin de los sentimientos, siendo el total la suma de la cantidad de sentimientos que poseen las palabras presentes en todos los captulos.

Se hace una proporcin debido a que, creemos que es la mejor mandera de representar las emociones en cada captulo, ya que, si solamente se cuentan las palabras, pueden ocurrir errores de interpretacin.

In [27]:
c=1
lib35_nrc = lib35_token %>% 
    inner_join(get_sentiments("nrc"))%>% #Al dataframe "lib35_token" se agrega el lxico "nrc"
    count(sentiment, chapter) %>% #Se cuentan la cantidad de sentimientos por captulos
    group_by(sentiment) %>%
    mutate(prop_percent = n/sum(n)*100, #se agrega la columna que posee la proporcion de cada sentimiento por captulo.
                                       #Se toma la cantidad de emociones en todos los captulos como el total.
          max = max(prop_percent)) #Se agrega una columna con la maxima proporcion presente en los sentimientos, para usarla en el siguiente ciclo.
    

for (i in lib35_nrc$prop_percent) { #ciclo para mostrar el captulo mas representativo de cada sentimiento.
    if (lib35_nrc$prop_percent[c] == lib35_nrc$max[c]){ #condicion, donde el valor maximo de la proporcion de los sentimientos, debe coinsidir con la proporcion del capitulo itinerado
        cat("El sentimiento", lib35_nrc$sentiment[c], "es mas representativo en el captulo", lib35_nrc$chapter[c], "\n")
    }
c=c+1
}
